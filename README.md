# Santa-City-Orion

-Santa City Orion on jouluinen versio Cyber City Orion -selainpelistä. Peli rakentuu käytännössä täysin samalla tavalla kuin alkuperäinen versio.
-Peli on kolmannessa persoonassa.
-Pelissä kerätään kolikoiden sijasta kirjeitä, mitä lapset ovat pukille kirjoittaneet. Vaikka kirjeitä on kolikoihin verrattuna vähän, ne ovat yhtä hankalasti saavutettavissa kuin alkuperäiset kahdeksan.
-Aina kun löytyy uusi kirje, avautuu uusi minipeli, joka liittyy kirjeessä olevaan lahjatoiveeseen. Minipelejä on neljä, kuten kirjeitäkin. 
-Peli toteutetaan Unityssä, kielenä C#. 
-Joulupukilla ei ole rekeä, vaan rakettireppu, joka näyttää ihan lahjakassilta. Siihen on kiinnitettynä kaksi sinistä kristallikuutiota, jotka mahdollistavat lentämisen ilman poroja. 
-Pelaaminen tapahtuu rakennusten katoilla.
